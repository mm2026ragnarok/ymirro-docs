# How to Install the Client

1. Download the YmirRO client from **https://ymir-ro.com/**.
2. Extract or install into a folder like `C:\Games\YmirRO`.
3. Ensure the folder is excluded from antivirus if patches fail.
4. Run the patcher as Administrator.
5. Wait until patching finishes, then **Start Game**.

If you encounter errors, see [Support & Contact](support-and-contact.md).
