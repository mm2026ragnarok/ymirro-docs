# System Requirements

YmirRO is lightweight and can run on most modern PCs.

## Minimum

- OS: Windows 7 / 8 / 10 / 11
- CPU: Dual-core
- RAM: 2 GB
- Storage: ~4 GB free
- Internet: Stable broadband
- GPU: Any card that supports DirectX 9+

## Recommended

- OS: Windows 10 / 11
- CPU: 4-core
- RAM: 4 GB+
- SSD for faster loading

For patching issues, whitelist the YmirRO folder in any antivirus/firewall.
