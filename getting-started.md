## 2. Create Your Game Account

1. Visit the website: **https://ymir-ro.com/**
2. Go to **Register** / **Create Account**.
3. Fill in your:
   - Username
   - Password
   - Email address
4. Confirm your registration (email verification if required).

You will use this account to log in to the game.

---

## 3. Install & Patch the Game

1. Run the installer or unzip the client into a folder (e.g. `C:\Games\YmirRO`).
2. Right-click the **patcher/launcher** and choose **Run as Administrator** (recommended).
3. Wait for all files to finish patching.
4. When the **Patch Complete** message appears, close and restart the patcher once to make sure everything is up to date.

---

## 4. Launch the Game

1. Open the YmirRO patcher.
2. Press **Start Game**.
3. Enter your YmirRO **Account ID** and **Password**.
4. Create your first character and start your adventure.
